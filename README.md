## How to Customize

### 1. Personal Information
Edit `index.html` and update:
- Your name in the hero section
- Professional title and description
- About section content
- Contact information (email, phone, location)

### 2. Skills
Update the skills section in `index.html`:
- Add or remove programming languages
- Adjust skill percentages
- Update technology tags

### 3. Projects
For each project, update:
- Project images (add images to the folder)
- Project names and descriptions
- Technology tags
- Links to live demos and GitHub repos

### 4. Experience
Update the timeline with your:
- Work experience
- Job titles and companies
- Dates and achievements
- Education details

### 5. Colors & Styling
Modify `styles.css` variables at the top:
```css
:root {
    --primary-color: #6c5ce7;
    --secondary-color: #a29bfe;
    --dark-color: #2d3436;
    /* ... other colors */
}
```

### 6. Social Links
Update all social media links in:
- Hero section
- Footer section

## Adding Images

Place your images in the project folder:
- `profile.jpg` - Your profile photo
- `project1.jpg`, `project2.jpg`, etc. - Project screenshots
- `resume.pdf` - Your CV/Resume

## Running the Portfolio

1. Open `index.html` in any modern web browser
2. Or use a local server:
   - Python: `python -m http.server 8000`
   - Node.js: `npx http-server`
   - VS Code: Use Live Server extension
